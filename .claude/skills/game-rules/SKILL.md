# 鳴き大富豪 ルールリファレンス

詳細なルールは `naki_daifugo_rules.md` を確認すること。

## コアルール早見表

### カードの強さ
- 通常: 3 < 4 < ... < A < 2 < Joker
- 革命時: 逆順（3が最強）、Jokerは常に最強
- Jokerを切る: ♠3単体のみ

### 鳴き（インターセプト）
- 条件: 単体出し、ランク6〜Q、同スートの前後カードを両方持っている
- 結果: 3枚セット → 場リセット → 鳴いたプレイヤーのターン
- 鳴けない: ペア/トリプル出し、ランク3-5/K/A/2、Joker、スート不一致

### 革命
- 同ランク4枚以上（Joker含む可）
- 同スート連番4枚以上（Joker含む可）

### ゲーム進行
- 初回: ♥7持ちが先攻
- パスしたら場が流れるまで出せない

## 実装時の注意
- shared/gameLogic.ts にルールロジック集約
- 連番判定は数値順（革命中でも変わらない）
